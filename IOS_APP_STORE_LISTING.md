## 📦 BUILD UPLOAD (su Mac)
```bash
# 1. Sul Mac, nella cartella del progetto:
npm install
npx cap sync ios
npx cap open ios

# 2. In Xcode:
# - Seleziona il target "App"
# - Tab "Signing & Capabilities" → seleziona il tuo Team Apple
# - Verifica Bundle ID: ai.econopulse.app
# - Versione: 1.0.0  Build: 1
# - Menu: Product → Archive
# - Window → Organizer → Distribute App → App Store Connect → Upload
```
